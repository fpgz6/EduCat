## 项目需求分析

### 基本项目业务逻辑

通过平台可以注册成为学生或者老师，学生可以通过平台报名参加自己想要学习的课程，报名分为付费报名和免费报名两种。付费报名是通过支付宝、微信支付。当课程报名成功之后，学员可以通过“我的课程”页面重新进入到课程的学习页面。老师可以通过平台创建课程，可以选择付费课程和免费课程，并且可对课程进行后续更新。老师获得的收入可在个人钱包页面查看并且提现。
